MyWeiBoDEMO
===========
###关于项目:
####1.这是一个仿造新浪微博的项目,利用新浪微博的开发接口来实现简单的查看微博和发送微博功能.
####2.不知道什么原因,一直无法直接push项目文件到Github上来.所以就把项目打包成了*__MyWeiBo.zip__*压缩包文件
